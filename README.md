# back-ollama-python-server

FastAPI 기반으로 Ollama LLM을 연동하는 백엔드 서버

---

## 🛠️ ENV (개발 환경)

- **Python**: 3.11.3
- **FastAPI**: 0.115.8
- **Uvicorn**: 0.34.0
- **Requests**: 2.32.3
- **Ollama**: 최신 버전 (`ollama serve` 실행)
- **Virtual Environment**: `venv` 사용 (`python -m venv venv`)
- **Package Manager**: `pip`
